# Contributing

This is a community-curated collection. Contributions welcome.

## Submit a New Tip

Open a PR or issue with:

- **Who said it** -- name, X handle, GitHub
- **What they said** -- the actual tip, quoted if possible
- **Source link** -- direct link to the tweet, thread, talk, or article
- **Which book** -- existing expert, or propose a new one

Tips must be about Claude Code specifically -- not general AI advice.

## Report a Correction

If something is wrong, outdated, or misattributed:

1. Open an issue using the "Correction" template
2. Include: which verse (e.g., Boris 3:4), what's wrong, what the correct info is
3. Link to docs or source that supports the correction

## Add a New Expert

To propose a new book:

1. The expert should have **significant public content** about Claude Code (not just one tweet)
2. They should bring a **distinct perspective** not already covered
3. Open an issue first to discuss before writing the full book
4. Follow the existing format: Chapter/Verse structure, source links on every tip

## Style Guidelines

- Keep the tone technical and direct -- the literary framing is in the structure, not the prose
- Every tip needs a source link
- Quotes should be verbatim when possible
- Don't editorialize -- let the expert's words speak

## Process

1. Fork the repo
2. Make your changes
3. Open a PR with a clear description
4. Maintainer reviews and merges
